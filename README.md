# Optional CCFB Modpack Add-Ons
**Add ons for the Minecraft Modpack**

## Bean's Battle music CCFB Compat
**Adds battle music to pokemon battle events.**

For server install in resourcepacks folder. For solo world install both datapacks and resourcepacks folder.

## Pokemusic-Enviroment Cobblemon Music Pack Compat
**Adds reactive pokemon background music.**

- Placed in resourcepacks folder.

-  [REACTIVE MUSIC REQUIRED](https://modrinth.com/mod/reactive-music) and its dependency [YetAnotherConfigLib (YACL)](https://modrinth.com/mod/yacl)
